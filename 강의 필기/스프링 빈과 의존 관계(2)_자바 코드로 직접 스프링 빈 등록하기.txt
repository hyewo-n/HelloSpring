> 스프링 빈과 의존 관계(2) - 자바 코드로 직접 스프링 빈 등록하기

- 하나하나 스프링에 직접 등록! -> 직접 설정 파일에 등록


- SpringConfig 클래스 생성
- @Configuration
- @Bean : 내가 스프링 빈을 등록할거야

# 각각의 장단점

-setter 인젝션의 단점
  : 누군가가 멤버 컨트롤러를 호출했을 때 그게 public으로 열려있음
  : 아무 개발자나 호출 가능
  : 중간에 잘못 바꾸면 문제가 생길 가능성이 있음

- 요즘 권장하는 스타일 : 생성자 주입

- 상황에 따라 구현 클래스를 변경해야 한다면
  설정을 통해 스프링 빈을 등록한다.
